# Portfolio Website

### Installation
Making own portfolio including SASS, AJAX, PHP, Vue and so on

### Contributing
1. create a new repo on Github
2. create folders and files in the new repo on VSCode
3. create HTML pages
4. build SASS regarding provided CSS
5. build AJAX

### Requirements
HTML
SASS
AJAX/JS
PHP
Vue.js

### Prerequisites
VS Code
